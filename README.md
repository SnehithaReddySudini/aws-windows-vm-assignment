# AWS Windows Virtual Machine Assignment

## Objective
Launch a Windows virtual machine in AWS Free Tier and access it remotely using RDP.

## Tools Used
- AWS EC2
- Windows Server
- Remote Desktop Protocol (RDP)

## Step 1: Login to AWS
Login to AWS Management Console and open EC2 service.

## Step 2: Launch Instance
Click Launch Instance and select Windows Server AMI.

## Step 3: Instance Type
Select t3.micro (Free Tier eligible).

## Step 4: Configure Security Group
Allow RDP traffic (Port 3389) from Anywhere.

## Step 5: Launch Instance
Create a key pair and launch the instance.

## Step 6: Connect to Instance
Use RDP client and connect using:
Username: Administrator
